# 弱点画像

> 由各教练在练习中发现并更新，协调器汇总分析。

## 状态定义

| 状态 | 含义 | 触发条件 |
|------|------|---------|
| **发现** | 刚发现的新弱点 | 教练在练习中首次标注 |
| **训练中** | 正在针对性训练 | 制定了训练计划并开始执行 |
| **待复查** | 训练后等待验证 | 连续 2 次通过相关练习 |
| **已改善** | 复查通过，但还不够稳定 | 复查时表现明显提升 |
| **已稳定** | 连续 3 周无复发 | 多次复查均正常 |
| **复发** | 曾改善但再次出现 | 已改善/已稳定的弱点再次出错 |

## 状态流转

```
发现 → 训练中 → 待复查 → 已改善 → 已稳定
                    ↑          ↓         ↓
                    └── 复发 ←─┴─────────┘
```
