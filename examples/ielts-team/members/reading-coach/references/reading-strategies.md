## 一、题型详细策略

### True / False / Not Given（判断题·事实）

**核心区分**：
- **True**：文中信息 = 题目信息（包括同义替换）
- **False**：文中信息明确与题目矛盾（说了相反的事）
- **Not Given**：文中完全没有提到这个信息（不能推断）

**最容易混淆的**：False 和 NG。

```
判断流程：
Step 1: 找题目关键词，定位原文
Step 2: 原文没提到这个信息 → Not Given（停止，不要尝试推断）
Step 3: 原文提到了：
  → 和题目一致（含同义替换）→ True
  → 和题目矛盾 → False
```

**中国考生高频错误**：
- 把 NG 答成 False（原文没说，但感觉应该是反的）
- 推断过度（原文给了一半信息，自己脑补另一半）
- 关键词没同义替换（题目说 "increase"，原文说 "grow"，以为没提到）

---

### Yes / No / Not Given（判断题·观点）

**和 T/F/NG 的关键区别**：这道题问的是**作者的观点/主张**，不是客观事实。

- **Yes**：作者认为题目说的是对的
- **No**：作者认为题目说的是错的，或持相反立场
- **Not Given**：作者没有表达对这个问题的观点

**关键词**：claim / argue / believe / suggest / according to the author / the writer states

**中国考生高频错误**：把 Y/N/NG 和 T/F/NG 混用。雅思会在同一套卷里用两个不同系统，要看清楚题目要求。

---

### 配对：标题配段落（Matching Headings）

**难点**：段落主旨常常模糊；标题数量比段落多（干扰项）。

```
解题步骤：
Step 1: 先读所有标题，理解每个标题的核心主题（不超过 3 秒/个）
Step 2: 读第一段的第一句 + 最后一句（中心句常在这里）
Step 3: 找段落中反复出现的概念 = 段落主题
Step 4: 匹配标题，排除已用过的
Step 5: 不确定的先跳过，做完后用排除法回来
```

**常见陷阱**：
- 标题里的词出现在段落里，但只是细节不是主旨 → 不要只看词汇，看主题
- 有两个标题都差不多 → 看哪个更准确，区分"全部"和"部分"

---

### 配对：信息配段落（Matching Information）

**和配对标题的区别**：找的是具体信息在哪一段，不是段落主旨。

```
解题步骤：
Step 1: 读题目，找出具体信息点（数字/人名/事件/研究发现）
Step 2: 扫描各段落，重点看段落开头和关键词密集处
Step 3: 定位后验证：确认是这道题要的信息，不是相似但不同的内容
```

**中国考生高频错误**：用关键词直接跳，没有验证 → 常选到相邻段落。

---

### 填空题（Summary / Notes / Table / Diagram Completion）

**铁律**：
1. 严格看字数限制："NO MORE THAN ONE WORD" vs "NO MORE THAN TWO WORDS AND/OR A NUMBER"
2. 答案必须是原文中的词，不能改词性或改写
3. 注意 A/AN 和冠词是否计入字数

```
解题步骤：
Step 1: 读题目，找出空格前后的上下文
Step 2: 判断答案词性（名词/动词/形容词/数字）
Step 3: 用上下文词定位原文段落
Step 4: 找到原文对应句，提取答案词
Step 5: 检查字数和词性
```

---

### 单选题（Multiple Choice）

**特点**：选项中一半对一半错，很容易被迷惑。

```
解题步骤：
Step 1: 读题干，找关键词定位原文
Step 2: 读完对应段落（通常不止一句话）
Step 3: 逐项排除：
  - 找原文依据，没有就排除
  - 答案里有"极端词"（always/never/all/none）要特别小心
Step 4: 选最接近原文的选项
```

**陷阱**：选项 A 和 B 都有原文提到，但 A 是细节，B 是主旨 → 通常选主旨。

## 二、定位技巧详解

### 关键词类型和扫描顺序

| 优先级 | 关键词类型 | 举例 | 理由 |
|-------|----------|------|------|
| 1（最快）| 专有名词 | Einstein / New York / UNESCO | 原文会原样出现，扫一眼就找到 |
| 2 | 数字/年份/百分比 | 1987 / 42% / £300 | 数字密集度低，扫描快 |
| 3 | 不常见词/专业词 | photosynthesis / GDP | 全文只出现一两次，快速定位 |
| 4（最慢）| 普通词的同义词 | important → significant | 需要同义意识，最费时 |

### 高频同义替换（雅思考点）

| 题目词 | 原文可能出现的替换 |
|-------|-----------------|
| increase / rise | grow, surge, soar, climb, go up |
| decrease / fall | drop, decline, reduce, plummet |
| important | significant, crucial, vital, key, essential |
| problem / issue | challenge, obstacle, difficulty, concern |
| show / prove | demonstrate, indicate, suggest, reveal, establish |
| research / study | investigation, survey, examination, analysis |
| different | diverse, various, distinct, contrasting |
| new | novel, innovative, unprecedented, emerging |

## 三、时间管理系统

### 标准分配

```
60 分钟，3 篇文章（各 13-14 道题）

建议分配：
  Passage 1：17 分钟（最简单，题目难度低，分数要拿稳）
  Passage 2：21 分钟（中等，配对题/填空常在这里）
  Passage 3：20 分钟（最难，有时间压力）
  答题卡/检查：2 分钟

时间警示线：
  30 分钟时：应该在做 Passage 2（如果还在 Passage 1，速度太慢）
  50 分钟时：应该在做 Passage 3（如果还在 Passage 2，立刻跳）
```

### 卡题策略

```
单题超过 2 分钟没有答案 → 立刻：
  1. 猜一个最有可能的选项（不要留空白）
  2. 做标记，有时间再回来
  3. 继续下一题

"回来" 的条件：
  → 做完全篇还有 3 分钟以上
  → 否则不回，保持猜的答案
```

### 速度问题根因诊断

| 症状 | 根因 | 解决 |
|------|------|------|
| 每篇用 25 分钟以上 | 逐词阅读，没有扫读 | 练 skimming（5 秒读段落主旨） |
| 定位很慢 | 关键词选得不好 | 练专有名词和数字优先 |
| 做 T/F/NG 特别慢 | 每题都从头找 | 题目是顺序出现的，顺序定位 |
| Passage 3 总没时间 | 前两篇超时 | 设置硬截止：30 分钟必须到 P2 |

## 四、中国考生阅读典型错误

| 错误类型 | 具体表现 | 根本原因 |
|---------|---------|---------|
| **推断过度** | 把 NG 选成 True 或 False | 用逻辑推断填补文中空缺 |
| **字面匹配** | 题目词和原文词不同就以为没找到 | 缺少同义替换意识 |
| **T/F/NG 边界混淆** | NG 答成 False | 没理解 NG 的定义（文中没提 ≠ 错） |
| **选项迷惑** | 单选选了细节而非主旨 | 没有回归题干问的是什么 |
| **时间失控** | P3 只有 10 分钟 | 前两篇没有时间意识 |
| **填空词性错** | 填了动词但需要名词 | 没有在填之前判断词性 |
| **超字数** | 填了 3 个词但限 2 个词 | 没看字数限制 |

## 五、分数段特征

| 分数 | 核心特征 |
|------|---------|
| 5.5 | 每篇错 5-7 道，时间常常不够，T/F/NG 胜率低 |
| 6.0 | 每篇错 3-5 道，时间勉强够，配对题不稳定 |
| 6.5 | 每篇错 2-3 道，时间基本够，主要问题是同义替换和细节判断 |
| 7.0 | 每篇错 1-2 道，速度流畅，NG 判断准确，偶有陷阱题失分 |
| 7.5+ | 接近满分，只在 P3 难题或特别刁钻的配对题上失分 |

**6.0 → 6.5 的关键**：同义替换意识 + T/F/NG 判断不推断

**6.5 → 7.0 的关键**：配对题做完不超时 + 细节和主旨不混淆
