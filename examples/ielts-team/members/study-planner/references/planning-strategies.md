# 备考策略和时间分配

> 学习规划师专属知识。不同备考周期的策略和时间分配规则。

## 不同备考周期策略

| 距考试 | 阶段名 | 策略 | 重点 |
|--------|--------|------|------|
| 3个月+ | 基础期 | 全面打基础 | 词汇、语法、各题型熟悉 |
| 2-3个月 | 突破期 | 分科突破 | 最弱科目集中训练 |
| 1-2个月 | 强化期 | 真题为主 | 剑桥真题限时练习 |
| 2-4周 | 冲刺期 | 模拟冲刺 | 全真模拟+查漏补缺 |
| 1周内 | 调整期 | 心态调整 | 轻量复习，保持状态 |

## 时间分配算法

### 基本规则

1. **补短板优先**：最弱科目分配最多时间
2. **维持强项**：强项不能完全放弃，至少20%时间
3. **每天词汇**：不论什么阶段，每天至少30分钟词汇
4. **每周模拟**：考前4周起，每周至少1次模拟考

### 分配公式

```
差距 = 目标分数 - 当前水平
权重 = 差距 / 总差距之和
分配时间 = 总可用时间 × 权重
最低保证 = 每科每周至少2次练习
```

### 示例

目标7.0，当前：L6.5 / R6.0 / W5.5 / S6.0
- 差距：L=0.5, R=1.0, W=1.5, S=1.0 → 总差距=4.0
- 权重：L=12.5%, R=25%, W=37.5%, S=25%
- 每周10小时 → L=1.25h, R=2.5h, W=3.75h, S=2.5h + 词汇3.5h

## 提分优先级（性价比排序）

1. **听力和阅读**：技巧型，短期提分最快（2-4周见效）
2. **写作Task 1**：有固定模式，容易提0.5-1分（2-3周）
3. **口语Part 1**：准备常见话题，容易稳定发挥（1-2周）
4. **写作Task 2**：需要积累，中期见效（4-8周）
5. **口语Part 2/3**：需要大量练习，长期见效（4-8周）

## 模拟考安排规则

| 阶段 | 频率 | 范围 | 目的 |
|------|------|------|------|
| 基础期 | 每2周1次 | 单科 | 评估起点 |
| 突破期 | 每周1次 | 单科或半模 | 检验训练效果 |
| 强化期 | 每周1次 | 全科 | 适应考试节奏 |
| 冲刺期 | 每2-3天 | 全科限时 | 考试状态调整 |

## 瓶颈识别规则

| 信号 | 判断 | 建议 |
|------|------|------|
| 连续3次同分 | 进入瓶颈 | 更换训练方法，或找对应教练做专项突破 |
| 某科长期未练 | 退化风险 | 安排至少1次练习 |
| 词汇测试正确率<60% | 词汇瓶颈 | 增加词汇时间，减少其他科目 |
| 写作TA持续低 | 审题/结构问题 | 不是多写，而是学结构 |
| 听力Section 3-4崩溃 | 学术听力瓶颈 | 精听+学术词汇双管齐下 |

## 心理管理建议

- 设定可达成的短期目标（每周提升一个小目标）
- 不要和别人比，只和上周的自己比
- 备考疲劳时适当休息1天，比硬撑效果好
- 模拟考不是为了"考好"，是为了发现问题
- 考前1天不要做新题，复习已有笔记即可
